# Finance Analytics Dashboard

## Overview

An interactive Finance Analytics Dashboard built using Power BI to analyze transactions, customer segments, taxes, fees, and overall financial performance.

## Features

* KPI Cards for Total Amount, Transactions, Fees, Tax, and Average Transaction Value
* Year-over-Year (YOY) and Previous Year (PY) analysis using DAX
* Drill-Through functionality for detailed transaction analysis
* Dynamic Titles and Dynamic Measures
* Custom Tooltip Pages
* Bookmark-based Reset Filters
* Interactive Slicers and Page Navigation

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling

## Dashboard Insights

* Monthly transaction trends
* Customer segment analysis
* State-wise performance analysis
* Transaction status analysis
* Gender-wise customer distribution
